# AI Toolbox

A toolbox for AI utilities and helpers.

## Installation

```bash
pip install ai-toolbox
```

## Usage

```python
from ai_toolbox import example_function

result = example_function()
print(result)
```

## Development

Install development dependencies:

```bash
pip install -e ".[dev]"
```

Run tests:

```bash
pytest
```
